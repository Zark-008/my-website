工作方式说明

全部通过网络远程完成。AI大模型+公开数据+网络搜索=完整建厂可行性研究。

  • AI大模型内置玻璃制造行业知识——窑炉技术路线、产能投资基准、能耗指标、主要设备商、全球竞争格局。模型可以直接输出框架性判断。
  • 美国能源信息署(EIA)提供各州工业天然气和电力价格——这是玻璃厂选址最重要的变量。
  • 上市公司公开披露（O-I Glass/Ardagh Group年报）提供行业盈利水平、产能数据、成本结构基准。
  • 美国环保署(EPA)和各州环境部门网站上公开的排放标准和许可流程。
  • 各州经济发展署网站上公开的税收优惠和招商引资政策。
  • 不替代实地考察——但如果客户需要实地考察，我们的报告可以作为考察前的全面预研，让客户带着精准问题去。

#模块核心调研问题数据来源

M3选址分析六大维度多州对比——①天然气价($/MMBtu)、②电价(¢/kWh)、③硅砂产地距离、④客户(灌装厂)距离、⑤物流(铁路/公路)、⑥劳动力成本和可用性 → 输出Top5候选州排名EIA天然气/电力价格数据 + USGS硅砂矿分布 + BLS劳动力统计 + 各州EDA网站
M4建厂投资(CAPEX)窑炉系统+成型设备(IS机)+退火炉+冷端设备+厂房+公用工程；10万/15万/20万吨三档CAPEX估算；中国设备 vs 欧洲设备对比设备商公开产品资料(Emhart/Bottero/山东三金) + 行业基准(GPI/IBISWorld) + O-I/Ardagh年报折旧数据反推
M5运营成本(OPEX)吨玻璃能耗(天然气+电力)；原料成本(硅砂/纯碱/石灰石/碎玻璃)；人工成本(时薪+三班倒编制)；吨玻璃总成本估算EIA能源价格 + USGS矿产价格 + BLS制造业工资 + 行业能效基准数据
M6环境法规EPA NSPS(40 CFR Part 60 Subpart CC)玻璃窑炉排放标准；Title V空气许可流程；NPDES废水许可；加州额外要求(Cap-and-Trade)EPA 40 CFR原文 + 各州环境质量部网站 + 环境法规解读文章
M7税收优惠各州企业所得税率+制造业减免；Property Tax Abatement；IDA补贴；联邦Opportunity Zone；培训补贴(Training Grant)；TIF融资各州EDA公开优惠政策 + CBRE等选址报告公开摘要 + 近期建厂案例数据库(Area Development杂志)
M8劳动力分析玻璃制造熟练工美国供给；各州工资水平；工会化程度(USW覆盖)；培训资源；中国技术团队派遣可行性(H-1B/L-1签证政策)BLS OES数据 + USW工会公开协议 + 社区学院玻璃培训项目 + USCIS签证政策
M9原料供应链美国硅砂主产区产量和价格；怀俄明天然碱纯碱优势；石灰石供应；碎玻璃(cullet)回收体系USGS Mineral Commodity Summaries + 矿企公开资料 + GPI回收数据
M10客户开发大型品牌(可口可乐/百威/雀巢)玻璃瓶采购模式——长期合同、天然气联动定价公式、供应商认证流程；精酿啤酒/烈酒需求增长上市公司年报供应链披露 + 行业分析文章 + 精酿啤酒协会(Brewers Association)数据
M11财务模型年产15万吨玻璃容器厂财务模型——CAPEX折旧(窑炉8-12年)、OPEX、吨价×产能利用率、EBITDA率、IRR、投资回收期O-I/Ardagh年报财务数据对标 + 行业基准 + 能源成本预测
M12风险矩阵天然气价格波动±30%影响；排放标准升级；设备进口关税；工会组织化；纯碱价格周期；需求衰退——每项可能性和影响度评估EIA期货曲线 + EPA政策追踪 + NLRB工会选举数据 + 历史价格波动分析

这份调研与滑板车/三轮车有一个本质不同：建厂是数百亿美元级别的资本决策，我们的线上调研提供的是投资前预研(pre-feasibility study)——帮助客户在花钱请工程公司做正式可行性研究之前，先回答"值不值得深入研究"。我们报告中的选址排名和CAPEX估算，可以帮客户节省几十万美元的工程公司概念设计费。
